# カップウィズハンドル検出ツール

株価チャートの「カップウィズハンドル（Cup with Handle）」パターンを自動検出するWebアプリです。

## 機能

- **単一銘柄分析**: 銘柄コードを入力してパターン判定・チャート表示
- **一括スクリーニング**: 日経225/S&P500主要銘柄を一括スキャンして候補一覧表示
- **対応市場**: 日本株（東証）/ 米国株（NYSE/NASDAQ）

## セットアップ

```bash
# 依存パッケージのインストール
pip install -r requirements.txt

# アプリ起動
streamlit run app.py
```

## 使い方

### 単一銘柄分析
1. サイドバーで「単一銘柄分析」を選択
2. 市場（日本株/米国株）を選択
3. 銘柄コードを入力（日本株: `7203`、米国株: `AAPL`）
4. チャートとスコアが表示される

### 一括スクリーニング
1. サイドバーで「一括スクリーニング」を選択
2. スキャン対象を選択（プリセットまたはカスタム入力）
3. 「スキャン開始」をクリック
4. スコア順にソートされた結果一覧が表示される

## パターン検出ロジック（8ステップ）

1. 左リム検出（局所高値の特定）
2. 下落フェーズ検出（15〜50%の下落）
3. 底の検出（U字型判定）
4. 回復フェーズ検出（出来高増加確認）
5. 右リム検出（左リムと±5%以内）
6. ハンドル検出（15%以内の調整）
7. ハンドル回復検出
8. ブレイクアウト判定

## スコアリング（0〜100点）

| 項目 | 配点 |
|---|---|
| カップ形状（U字度） | 20点 |
| カップの深さ | 15点 |
| 左右リムの対称性 | 15点 |
| カップの期間 | 10点 |
| ハンドルの深さ | 15点 |
| ハンドルの出来高減少 | 15点 |
| ブレイクアウト出来高 | 10点 |
